<--------   Technologies de programmation utilisées   --------->

Pour réaliser ce projet, les technologies utilisées sont : HTML, Bootstrap, PHP, Javascript, JQuery, Ajax, Mysql


<--------   Connexion à la base de données   --------->

Les tests se font en Local.
Le fichier de connexion à la base de données se trouve dans le dossier "treatment".
La base de données exportée se trouve dans le dossier "bdd".


<--------   Explication brève du projet   --------->

La page d'accueil regroupe tous les modules. Au clic sur la simulation, elle se lance et toutes les secondes, des données sont 
envoyées et enregistrés dans la base de données.
Si un module surchauffe, une alerte s'affiche et le module s'arrête.

Le dashbard permet de voir à l'aide d'un graphique l'évolution des valeurs mesurées durant les simulations.
